Projects

Selected engineering work

Environmental IoT

CLEAR Environmental Monitoring Platform

Developed a cloud-connected environmental intelligence platform for monitoring CO₂, particulate matter, VOC index, NOx index, temperature, humidity, and noise. The system uses embedded sensing hardware, MQTT telemetry, dashboards, alerts, and analytics to support indoor and industrial air quality decisions.

  • Integrated SEN66, SPS30, and INMP441 sensors with ESP32-S3 hardware.
  • Implemented telemetry, dashboard visualisation, and remote device interaction.
  • Supported hardware optimisation, enclosure refinement, and deployment readiness.
Machine Monitoring

Current Sensor Utilisation System

Built an ESP32-based current monitoring system for tracking machine utilisation, threshold events, and runtime behaviour. The system supports ThingsBoard telemetry and two-way dashboard control through RPC and shared attributes.

  • RMS current measurement using EmonLib.
  • Remote threshold and sampling-rate configuration.
  • Designed for industrial asset usage tracking and operational insight.
Structural Monitoring

StrucSense Concrete Monitoring Device

Developed a wireless structural monitoring system for concrete structures using environmental and tilt sensing. The device measures temperature, humidity, acceleration, pitch, roll, and battery voltage, then sends data through WiFi or LoRa.

  • MPU6050 tilt measurement with baseline correction.
  • SHT31 temperature and humidity monitoring.
  • Battery-powered operation with deep-sleep strategy.
Industrial Test Bench

Motor, Vibration & Flow Monitoring Rig

Worked on a motor-driven pulley test setup for inducing faults and collecting operational data. The system included vibration monitoring, current measurement, flow sensing, temperature sensing, and cloud telemetry.

  • Integrated ADXL345 and industrial vibration sensors.
  • Worked with VFD-driven motor systems.
  • Used data collection for anomaly detection and AI training.
Mechanical Design

SolidWorks Enclosure Design

Designed multiple enclosures for CLEAR and smaller connected-device projects, focusing on manufacturability, mounting, accessibility, cable routing, sensor exposure, and visual alignment with product identity.

  • Electronics enclosure layouts for sensor devices.
  • Design-for-assembly and design-for-manufacturing improvements.
  • Product-ready housing concepts for field deployments.
